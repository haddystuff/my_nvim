# My nvim config
My neovim config

## Works with nvim 0.9

## Install

1. Install packages
```
python3-lsp-server
python3-lsp-black
pyright
flake8
gopls
ansible-lint
nerd-devicons
python-black
```
for example in Ubuntu you will need to run:
```sudo apt-get install python3-pylsp  flake8 gopls ansible-lint python3-pylsp-black black```

2. Install Nerd fonts

3. Install vim Plug

4. Coopilot config
